# Madrilla Launcher v1.0.0

Download the latest release zip from the Releases tab to get started.

### Features
This is a heavily modified and improved version of the "femboy lose" and "femboysense" launchers. It includes significantly more features for your software stack.

### Transparency & Reverse Engineering
This launcher and its components feature **absolutely zero obfuscation techniques or packing/protection**. 

The UI and routing logic are completely open-source (you are reading the source code repository right now). For any of the internal pre-compiled modules (like the `.dll` or `.exe` files included in the release zip), you are actively encouraged to reverse engineer them if you'd like to verify their safety or see how they tick. Everything is out in the open.
