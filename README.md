# Madrilla Launcher v1.0.0

This is a heavily modified and improved version of the "femboy lose" and "femboysense" launchers. It includes significantly more features for your software stack.

## Installation & Setup

To get started, follow these simple steps:

1. **Download:** Go to the [Releases](https://github.com/swastikaspammer-hue/Madrilla-Launcher/releases/tag/v1.0.0) tab and download `Madrilla-Launcher-Release.zip`.
2. **Extract:** Extract the downloaded `.zip` file into a folder of your choice.
3. **Setup Shortcuts (Optional):** Double-click `Setup.bat`. This will automatically create clean shortcuts on your **Desktop** and in your Windows **Start Menu** so you can easily search for the launcher.
4. **Launch:** Double-click `Launch.bat` (or use the created shortcuts) to start the launcher!

This launcher and its components feature **absolutely zero obfuscation techniques or packing/protection**.
